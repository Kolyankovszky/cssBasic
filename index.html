Szintaxis / Syntax

A CSS -szabály egy választóból,
selector és egy deklarációs blokkból, declaratin áll.

Például jelen esetben: body {background-color: black;}

Láthatod ebben a formában is:
body{
background-color: black;
}

A választó a stílusolni kívánt HTML elemre mutat, például "body".

body{<-ide, a kapcsos zárójelek közé írjuk a deklarációt->}

A deklarációs blokk egy vagy több pontosvesszővel elválasztott
deklarációt tartalmaz.

background-color: black;

Minden deklaráció tartalmaz egy CSS tulajdonságnevet
(pl: background-color:) kettősponttal elválasztva
és egy értéket (pl: black;).

Több CSS -deklarációt pontosvesszővel választanak el egymástól,
a deklarációs blokkokat pedig kapcsos zárójelek veszik körül.

Választó / Selectors

A CSS-szelektorok a stílusozni kívánt HTML-elemek „megkeresésére”
(vagy kijelölésére) szolgálnak.

A CSS-szelektorokat öt kategóriába sorolhatjuk:

- Egyszerű kiválasztók
(elemek kiválasztása név, azonosító, osztály alapján)
- Kombinátorválasztók
(az elemek kiválasztása a köztük lévő konkrét kapcsolat alapján)
- Pszeudoosztály szelektorok
(elemek kiválasztása egy bizonyos állapot alapján)
- Pszeudoelem-választók
(egy elem egy részének kijelölése és stílusa)
- Attribútumválasztók
(elemek kiválasztása attribútum vagy attribútumérték alapján)

A CSS elemválasztó

Az elemválasztó az elem neve alapján választja ki a HTML elemeket.
Példa:
Itt az oldal összes "p" eleme középre igazítva lesz,
piros szövegszínnel:
p{text-align: center; color: red;}

Láthatod ebben a formában is:
p{
text-align:>center;
color:red;
}

A CSS azonosító választó

Az azonosítóválasztó egy HTML elem id attribútumát használja
egy adott elem kiválasztásához. Egy elem azonosítója egyedi az oldalon belül,
így az id választó egy egyedi elem kiválasztására szolgál!
Egy adott azonosítójú elem kiválasztásához írjon be egy hash (#) karaktert,
amelyet az elem azonosítója követ.
Példa
Az alábbi CSS-szabályt alkalmazzuk az id="para1" HTML-elemre:
#para1{text-align: center; color: red;}

Láthatod ebben a formában is:
#para1{
text-align:center;
color:red;
}

A CSS osztályválasztó

Az osztályválasztó kiválasztja a HTML-elemeket egy adott osztályattribútummal.
Egy adott osztályhoz tartozó elemek kiválasztásához írjon be egy pontot (.),
majd az osztály nevét.
Példa
Ebben a példában az összes class="center" karakterláncú HTML-elem piros
és középre igazított lesz:
.center{text-align: center; color: red;}

Láthatod ebben a formában is:
.center{
text-align:center;
color:red;
}
Példa
Ebben a példában csak a "p" class="center" elemek lesznek piros és
középre igazítotva:
p.center{text-align: center; color: red;}
Láthatod ebben a formában is:
p.center{
text-align:center;
color:red;
}

A CSS univerzális választó

Az univerzális választó (*) kijelöli az összes HTML elemet az oldalon.
Példa
Az alábbi CSS-szabály hatással lesz az oldal minden HTML-elemére:
*{text-align: center; color: blue;}
Láthatod ebben a formában is:
*{
text-align:center;
color:blue;
}

A CSS csoportosítási választó
A csoportosítási választó kiválasztja az összes HTML-elemet azonos stílusdefiníciókkal.
Nézze meg a következő CSS kódot (a h1, h2 és p elemek azonos stílusdefiníciókkal rendelkeznek):

h1{text-align: center; color: blue;}

h2{text-align: center; color: blue;}

p{text-align: center; color: blue;}

Jobb lesz csoportosítani a kiválasztókat, minimalizálni a kódot.
A kijelölők csoportosításához vesszővel válassza el az egyes választókat.
Példa
Ebben a példában a fenti kódból csoportosítottuk a kiválasztókat:

h1, h2, p {text-align: center; color: blue;}

Láthatod ebben a formában is:

h1, h2, p {

text-align: center;

color: blue;

}

Megjegyzés / Comments

CSS megjegyzések

A CSS megjegyzések nem jelennek meg a böngészőben, de segíthetnek dokumentálni a forráskódot.
A megjegyzések a kód magyarázatára szolgálnak, és segíthetnek a forráskód későbbi szerkesztésénél.
A böngészők figyelmen kívül hagyják a megjegyzéseket.
Az style elemen belül egy CSS megjegyzés kerül elhelyezésre, amely a következővel kezdődik /* és ezzel végződik */:

Példa:
/* This is a single-line comment */
p {
color: red;
}

Szín / Color

CSS színek
A színek megadása előre meghatározott színnevek vagy RGB, HEX, HSL, RGBA, HSLA értékek használatával történik.

CSS színnevek
A CSS-ben a szín egy előre meghatározott színnév használatával adható meg, CSS/HTML 140 szabványos színnevet támogat.
red, yellow, brown, blue, white, black, ...

CSS színértékek
A CSS-ben a színek RGB-, HEX-, HSL-, RGBA- és HSLA-értékekkel is megadhatók.

Megegyezik a "paradicsom" színnévvel:
rgb(255, 99, 71)
vagy
#ff6347
vagy
hsl(9, 100%, 64%)

Hol használjuk?

CSS háttérszín (background-color), beállíthatja a HTML elemek háttérszínét

body{background-color: black;}

CSS szöveg színe (color), beállíthatja a szöveg színét

h1{color: yellow;}

CSS szegélyszín (border), beállíthatja a szegélyek színét.

h1{border: red;}

Háttér / Backgrounds

CSS hátterek
A CSS-háttér tulajdonságai az elemek háttéreffektusainak hozzáadására szolgálnak.

  1. background-color
  2. background-image
  3. background-repeat
  4. background-attachment
  5. background-position
  6. background (rövid tulajdonság)

CSS háttérszín
A background-colortulajdonság egy elem háttérszínét adja meg.
Példa
Az oldal háttérszíne a következőképpen van beállítva:

body{background-color: black;}


A CSS-ben a színt leggyakrabban a következők határozzák meg:
érvényes színnév - például "piros"
HEX érték – például "#ff0000"
RGB-érték – például „rgb(255,0,0)”


CSS háttérkép
A background-image tulajdonság egy elem háttereként használandó képet határoz meg.
Alapértelmezés szerint a kép ismétlődik, így a teljes elemet lefedi.
Példa:
body{background-image: sarkany.jpg;}

Megjegyzés: Ha háttérképet használ, olyan képet használjon, amely nem zavarja a szöveget.

A háttérkép bizonyos elemekhez is beállítható, például a p elemhez:
p{background-image: sarkany.jpg;}


CSS- háttérkép ismétlése
Alapértelmezés szerint a background-imagetulajdonság vízszintesen és függőlegesen is megismétli a képet.
Egyes képeket csak vízszintesen vagy függőlegesen kell megismételni, különben furcsán néznek ki.
Tipp: Egy kép függőleges ismétléséhez állítsa bebackground-repeat: repeat-y;
CSS háttérkép ismétlés lehetősége a nincs ismétlés.
A háttérkép csak egyszeri megjelenítését is meghatározza a background-repeattulajdonság.
Példa:
Csak egyszer jelenítse meg a háttérképet:
body {
background-image: sarkany.jpg; background-repeat: no-repeat;
}


CSS háttérpozíció
A background-positiontulajdonság a háttérkép pozíciójának megadására szolgál.
Példa:
Helyezze el a háttérképet a jobb felső sarokban:
body {
background-image: sarkany.jpg;
background-repeat: no-repeat;
background-position: right top;
}


CSS háttérmelléklet
A background-attachment tulajdonság megadja, hogy a háttérképet görgetni kell-e vagy rögzíteni kell (nem fog gördülni az oldal többi részével).
Példa:
Adja meg, hogy a háttérképet rögzíteni kell:
body {
background-image: sarkany.jpg;
background-repeat: no-repeat;
background-position: right top;
background-attachment: fixed;
}

Adja meg, hogy a háttérképnek az oldal többi részével együtt kell gördülnie:
body {
background-image: sarkany.jpg;
background-repeat: no-repeat;
background-position: right top;
background-attachment: scroll;
}


CSS-háttér – Gyorsírás tulajdonság
A kód lerövidítéséhez lehetőség van az összes háttértulajdonság megadására egyetlen tulajdonságban. Ezt gyorsírási tulajdonságnak nevezik.
Példa, ahogy eddig írtuk:
body {
background-image: sarkany.jpg;
background-repeat: no-repeat;
background-position: right top;
background-attachment: fixed;
}

Példa, gyorsírás tulajdonsággal, background:
body {
black sarkany.jpg no-repeat fixed right top;
}

A gyorsírás tulajdonság használatakor a tulajdonságértékek sorrendje a következő:

  1. background-color
  2. background-image
  3. background-repeat
  4. background-attachment
  5. background-position

Nem számít, ha valamelyik ingatlanérték hiányzik, ha a többi ebben a sorrendben van.
